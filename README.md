# Assistente Financeiro

Uma aplicação web moderna e responsiva para controle financeiro pessoal.

## Funcionalidades

- **Dashboard**: Visualização rápida do Saldo Atual, Receitas e Despesas.
- **Gráficos**: Gráfico de rosca para analisar a proporção de Entradas vs Saídas.
- **Transações**: Histórico detalhado de todas as movimentações.
- **Cadastro**: Adicione receitas e despesas facilmente através de um formulário intuitivo.
- **Responsivo**: Funciona perfeitamente em computadores e celulares.
- **Persistência**: Seus dados ficam salvos no navegador.

## Como usar

1.  Instale as dependências (se ainda não fez):
    ```bash
    npm install
    ```

2.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

3.  Abra o navegador no endereço indicado (geralmente `http://localhost:5173`).

## Tecnologias

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Recharts](https://recharts.org/) (Gráficos)
- [Lucide React](https://lucide.dev/) (Ícones)
- CSS Modules & Glassmorphism UI
